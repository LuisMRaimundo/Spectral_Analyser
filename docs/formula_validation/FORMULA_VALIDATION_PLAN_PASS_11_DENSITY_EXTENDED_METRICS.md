# Formula Validation Plan — Pass 11 — Density Extended Metrics

## 1. Scope

This plan covers hand-checkable numerical fixtures for the **non-ambiguous** formulas documented in `docs/formula_extraction/FORMULA_EXTRACTION_TABLE_PASS_11_DENSITY_EXTENDED_METRICS.md`, i.e. extended **`density.py`** metrics from Pass 11 (noise floor, physical spectral density, Bark map, harmonic / inharmonic density and richness, harmonic effective power descriptors, sub-bass protection tolerance, low-frequency residual aggregation, partial-density bundle, combined density metric, and a **small** `spectral_density` slice). It is intended so the same cases can later be turned into `pytest` tests; this file does **not** implement tests.

## 3. Excluded or deferred cases

Deferred until human review or a broader Pass 11 bis (not asserted as settled truth):

- **`calculate_perceptual_spectral_density`** — masking loop and inverse Bark-centre heuristic for band-level dB comparisons.
- **`estimate_noise_floor_by_critical_bands`** — boundary interpolation between adjacent critical-band noise floors.
- **`apply_spectral_masking_filter`** — full iterative audibility / sorted masker loop.
- **`spectral_density`** — optional **`D_harm`** tail (depends on `apply_density_metric` and `f0_hz`), and the **\(M>1000\)** sparse proximity approximation path.
- Any ambiguous rows in `FORMULA_EXTRACTION_TABLE_PASS_11_DENSITY_EXTENDED_METRICS.md` that were explicitly flagged for human review.
